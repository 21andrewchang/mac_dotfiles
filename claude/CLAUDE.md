# CLAUDE.md (global)

## Response length

- **Length target:** aim for ~120 words per reply (~800 chars). Soft target, not a cap — finish the thought rather than truncate, but don't pad past it. One-line answers stay one line.
- **Artifacts over walls of text:** if a reply would run past ~250 words or needs real structure (tables, multi-section, code blocks), put it in an artifact and leave a 1–2 line summary in the terminal.
